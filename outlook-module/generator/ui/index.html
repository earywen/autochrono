ETAPE 1 : Generer les fichiers

ETAPE 2 : Ouvrir l'editeur VBA

1. Ouvrez Outlook

2. Appuyez sur Alt + F11

ETAPE 3 : Importer le module

1. Fichier ‚Üí Importer un fichier...

2. Selectionnez ChronoCreator.bas (sur votre Bureau)

ETAPE 4 : Coller le code Session

1. Double-cliquez sur ThisOutlookSession (panneau gauche)

2. Cliquez dans la zone blanche

3. Ctrl + V pour coller

4. Ctrl + S pour sauvegarder

5. Fermez l'editeur (croix rouge)

ETAPE 5 : Ajouter le bouton

1. Allez dans Fichier > Options > Personnaliser le ruban

Astuce : Pour le mettre dans la fenetre "Message", faites ceci depuis un Nouveau Message !

2. A droite, choisissez l'onglet (ex: Accueil ou Message)

3. Cliquez Nouveau groupe ‚Üí Renommez "Chrono"

4. Liste "Choisir les commandes" (gauche) : choisissez Macros

5. Selectionnez NouveauChrono

6. Cliquez Ajouter >>

7. Cliquez Renommer pour changer l'icone/nom

8. Validez par OK
